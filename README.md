# NeuroVive Application 

A mobile application made to facilitate the early detection of the Parkinson disease, using our AI model.

## About The Application 

There are 3 main steps in the use flow, 

1. Voice Recording test
2. HandWriting Spiral test
3. Additional info from our smart pin used in the handwriting 

### Key Features 
- **Voice Recording**: A dedicated step for users to provide a voice sample for analysing 
- **Handwriting**: A dedicated step for users to provide handwriting tests for analysing.(Still in development)
- **Clean Results Screen**: A screen to display the processed results cleanly and quickly.
- **Cross-Platform**: Built with Flutter to support all major desktop and mobile platforms.

## Getting Started

Follow these steps to get a local copy up and running, or you can simply download from the releases

 ### Prerequisites
- Flutter SDK: [Install Flutter](https://docs.flutter.dev/get-started/install)
- An IDE (Android Studio, VS Code, or Xcode for iOS/macOS)

 ### Installation
1.  Clone the repository:
    ```sh
    git clone https://github.com/NeuroVive/NeuroVive-Mobile-Application-
    ```
2.  Navigate to the project directory:
    ```sh
    cd [project name]
    ```
3.  Fetch the dependencies:
    ```sh
    flutter pub get
    ```
4.  Run the app:
    ```sh
    flutter run
    ```
